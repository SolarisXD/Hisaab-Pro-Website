# Hisaab-Pro Website

Welcome to the official landing page repository for **[Hisaab-Pro](https://github.com/SolarisXD/Hisaab-Pro)**, a professional-grade, double-entry business ledger system designed specifically for retail shops and small businesses. 

This repository contains the source code for the promotional website, built to be incredibly fast, responsive, and aesthetically premium.

![Hisaab-Pro Website Preview](/public/setup-wizard%20page%203.png)

## ✨ Overview

Hisaab-Pro is an offline-first, USB-portable accounting application that ensures mathematical precision and strict trial balance integrity using AES-256 encryption. This website serves to promote the app, providing feature breakdowns, dynamic GitHub release fetching, and issue reporting.

**Key Website Features:**
- **Live GitHub API Integration:** Automatically fetches the latest stable release binaries and changelogs directly from the main Hisaab-Pro repository.
- **Glassmorphic UI:** Modern, premium aesthetics featuring tailored typography, deep dark modes, and subtle micro-interactions.
- **Framer Motion:** High-fidelity, smooth scrolling animations and element transitions.
- **Static Generation (SSG):** Fully optimized using Next.js for sub-second load times and flawless SEO.
- **Integrated Issue Reporting:** A streamlined issue reporting form that constructs and forwards fully-formatted Markdown bug reports directly to GitHub.

## 🛠️ Technology Stack

- **Framework:** [Next.js 14+ (App Router)](https://nextjs.org/)
- **Library:** [React 18](https://react.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Deployment:** Vercel / Netlify Ready

## 🚀 Getting Started

To run the development server locally:

```bash
# 1. Clone the repository
git clone https://github.com/SolarisXD/Hisaab-Pro-Website.git

# 2. Install dependencies
npm install
# or yarn install / pnpm install

# 3. Run the development server
npm run dev
# or yarn dev / pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
